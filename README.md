# json-to-model README

"json-to-model". json格式数据转模型文件\n
目前只支持json 转 dart

<br>使用方法：
<br>1、新建模型文件
<br>2、输入json数据如：
<br>{<br>  "code":0,
    <br>"message":"successful",
    <br>"data":{
        <br>"name":"qinkuan",
        <br>"age":1
        <br>}
<br>}
<br>3、右键选择json转模型文件，或快捷键mac: command+3 win:ctrl+f3即可
